# Simple Inventory Management API

A beginner-friendly backend project built using Node.js and Express.js.  
This application performs basic CRUD operations for an inventory system using a JSON file.

## Features
- Get all products
- Add a new product
- Update product description
- Delete a product

## Technologies Used
- Node.js
- Express.js
- JavaScript
- JSON

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start server using `node index.js`
4. Test APIs using Postman

## API Endpoints
- GET `/getProducts`
- POST `/addProduct`
- PUT `/updateProduct`
- DELETE `/deleteProduct`
